# PyGameOfLife

## Requirements

You need to have installed:
- Python
- pygame
- numpy

--- 
## Game

    Press ESC or F4 to quit the game
    Press RETURN to restart the game
    Press SPACE to stop or resume the game
    Press one of the letter below to change the color of the alive cells:
        - r: red
        - b: blue
        - g: green
        - c: cyan
